## 1. Project Overview

packagingconveyor.com is a lead-generation website for the packaging conveyor equipment industry. The site serves as a neutral resource helping packaging engineers, plant managers, and operations teams select the right conveyor systems for their production lines.

The site contains 51 pages organized into four pillars: Conveyor Types (equipment-focused), Industry Applications (food, packaging, pharma), Conveyor Functions (task-based: accumulate, divert, load, stack), and Buyer Resources (directory, design guides, automation assessment).

Every page needs visual content. This document specifies exactly what each illustration should contain, the viewing angle, required callouts, dimensions, and production notes. The goal is to produce a consistent visual library that works across the website, YouTube videos, PDF downloads, and social media.

## 2. Visual Style Guide

### 2.1 Illustration Style

**Primary style: Isometric technical line art.** Clean vector illustrations with consistent viewing angles. Think technical manual meets modern infographic. Not photorealistic. Not cartoon. Professional, accurate, and clear.

**Line weight:** 2px primary outlines, 1px internal detail lines, 0.5px dimension/callout lines. Consistent across all illustrations.

**Viewing angle:** Default isometric (30-degree) for equipment overview shots. Side cross-section for internal mechanism views. Top-down for layout and flow diagrams. Each illustration spec below indicates which view to use.

### 2.2 Color Palette

| Element | Color | Usage |
|---------|-------|-------|
| Conveyor Frame / Structure | #1E3A5F (Navy Blue) | Primary structural elements |
| Rollers / Moving Parts | #94A3B8 (Slate Gray) | Rollers, chains, belts in neutral |
| Products / Loads | #F59E0B (Amber) | Boxes, pallets, food items on conveyors |
| Callout Lines | #3B82F6 (Blue) | Leader lines from labels to components |
| Callout Labels | #0F172A (Near Black) | Text labels, always on white/light background |
| Motion / Flow Arrows | #EF4444 (Red) | Direction of product flow, movement paths |
| Safety / Guards | #FCD34D (Yellow) | Safety guards, barriers, emergency stops |
| Sensors / Controls | #22C55E (Green) | Photo-eyes, PLCs, control elements |
| Background | #FFFFFF (White) | Always white background for web/print flexibility |
| Section Highlighting | Varies by context | Use 10% opacity fills to highlight zones or sections |

### 2.3 Callout Style

**Leader lines:** Thin blue lines (1px) from component to label. Use elbowed lines (horizontal + vertical segments), not diagonal. Small filled circle (4px) at the component anchor point.

**Label text:** Arial or DM Sans, 10–12px, near-black (#0F172A) on white pill-shaped background with subtle border. All labels outside the illustration boundary when possible to keep the drawing clean.

**Dimension lines:** Use sparingly and only when spec values are shown (roller pitch, belt width, etc.). Standard dimension line style with arrows on both ends.

### 2.4 File Delivery Requirements

**Format:** SVG (primary) + PNG at 2x resolution (secondary). SVG enables web scaling and future animation. PNG for immediate use in PDFs and social media.

**Layers:** If delivering AI/SVG, use named layers: Structure, Moving Parts, Products, Callouts, Arrows, Background. This enables animation and selective use.

**Artboard size:** Match the dimensions specified in each illustration brief below. All illustrations are designed for 1200px content width on the website.

**Naming convention:** `illustration-[page-slug]-[descriptor].svg` — Example: `illustration-roller-conveyors-cross-section.svg`

## 3. Asset Type Definitions

### 3.1 Custom Technical Illustrations (~30 total)

Commissioned from illustrator. Cross-sections, cutaways, isometric views showing how equipment works. These are the hero images for conveyor type pages and function pages. Each brief below includes exact requirements.

### 3.2 Data Visualizations (~10 total, built in-house)

Comparison matrices, selection flowcharts, spec tables, interactive calculators. Built in code (HTML/CSS/JS) or Figma. No illustration commission needed. These live on category pages, directory pages, and the design guide.

### 3.3 Flow Diagrams (~6 total)

Process flow diagrams showing product path through packaging lines. Boxes-and-arrows style with conveyor type labels. Simpler than technical illustrations. Can be commissioned cheaply or built in Figma/draw.io.

### 3.4 Icon Set (1 commission, ~25 icons)

Simple conveyor type icons at ~200x200px each. Used across the site for navigation cards, comparison grids, and category page selectors. One consistent isometric style. Commission as a set from the same illustrator doing the technical drawings.

## 5. Reference Sources for Illustrators

These sources provide mechanical accuracy references. You are NOT copying these. You are using them to understand the mechanical reality of each conveyor type so your illustrations are technically accurate.

### 5.1 Manufacturer Catalogs (Free PDFs)

**Dorner** (dornerconveyors.com) — Clean product photos and line drawings. Good reference for flat belt, modular, and sanitary conveyors.

**Hytrol** (hytrol.com) — Engineering catalogs with cross-sections for gravity roller, powered roller, belt conveyors. PDF downloads available.

**Interroll** (interroll.com) — MDR (motorized driven roller) cutaway diagrams. Best reference for internal motor detail.

**Rexnord/Regal Rexnord** (regalrexnord.com) — Chain, tabletop, and modular belt technical drawings.

**KWS Manufacturing** (kwsmfg.com) — Screw conveyor and bucket elevator engineering guides with detailed cross-sections. Their engineering guide PDF is the gold standard for screw conveyor diagrams.

**Wire Belt Company** (wirebelt.com) — Wire mesh belt weave patterns and conveyor applications in food processing.

### 5.2 Google Patents

Search patents.google.com for any conveyor type. Patent drawings are public domain and show precise mechanical detail. They are ugly but structurally accurate. Use these to verify your mechanical understanding of how each component works.

### 5.3 Competitor Websites

**Titan Conveyors** (titanconveyors.com) — Product photos for CDLR, wire mesh, hinged steel belt, cleated. Their top traffic pages.

**PFI Conveyors** (pficonveyors.com) — Food industry conveyors, vibratory, storage/accumulation. Bakery-focused.

**KWS Manufacturing** (kwsmfg.com) — Screw conveyor engineering guides, component diagrams.

## 7. YouTube & Animation Considerations

All illustrations will be repurposed for a faceless YouTube channel. When producing the illustrations, keep the following in mind for future animation:

**Layered files:** Deliver SVG or AI with named layers. A motion graphics artist will need to animate individual components (rollers spinning, products moving, zones activating). If everything is flattened, animation requires redrawing.

**Separate moving parts:** Products, conveyor belts, chains, rollers, robot arms should each be on their own layer. Static elements (frame, floor, labels) on a separate background layer.

**Motion paths:** Where product flow arrows are shown, consider drawing the product in multiple positions along the path (3–4 positions) on separate layers. This gives the animator keyframes to work with.

**Consistent canvas size:** YouTube is 1920×1080. All illustrations should be designed to fill this frame well. The web dimensions (1200px wide) will be scaled up. Ensure detail holds at 1920px width.
