# packagingconveyor.com — Project Data

Drop this folder into your Claude Code project as reference data.

## Files

### Strategy Docs
- `packagingconveyor_ia.md` — Full 51-page information architecture with URLs, H1s, content briefs, keyword data, visual specs, EMD redirect map, and phased build order. **This is the primary reference for building the site.**
- `packagingconveyor_illustration_spec.md` — Illustration PRD with style guide, color palette, 15 detailed briefs, reference sources, budget, and YouTube/animation notes.

### GSC Data (4 ProMach conveyor brand properties, 16 months)

Combined: ~2M conveyor-related impressions across all 4 properties.

- `gsc_shuttleworth.csv` — Shuttleworth (accumulation, product handling, indexing). 1.2M conveyor impressions. Top: "indexing conveyor" 14K imp, "accumulation conveyor" 12.7K imp, "stainless steel conveyor" 11.7K imp.
- `gsc_benchmark.csv` — Benchmark Automation (feeding, loading, distribution). 298K conveyor impressions. Top: "automatic tray loader" 4.3K imp, "tray loading" 3.2K imp, "stacking machine" 2.6K imp.
- `gsc_kleenline.csv` — KLEENLine (sanitary, washdown, food-grade). 293K conveyor impressions. Top: "trough conveyor" 8.1K imp, "sanitary conveyors" 3.9K imp, "diverting conveyor" 3K imp, "washdown conveyor" 2.5K imp.
- `gsc_statera.csv` — Statera (turnkey conveyor systems, pallet/case). 163K conveyor impressions. Top: "pallet conveyor" 18K imp, "case conveyor" 5.1K imp, "packaging conveyor systems" 2.6K imp.

## How to Use

Point Claude Code at `packagingconveyor_ia.md` as the site blueprint. The GSC files are the secret weapon — they show real impression data from 4 ProMach conveyor brands that Ahrefs massively underreports. Cross-reference GSC impressions vs Ahrefs volume to find opportunities where competition looks low in Ahrefs but actual search demand is 2-10x higher.
